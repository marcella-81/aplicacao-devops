# Aplicação DevOps

Projeto desenvolvido para a disciplina de DevOps.

## Integrantes

- Marcella Cagliari
- Paula Haas

## Tecnologias

- Node.js
- Express.js
- Git / GitHub

### Instalação
Clone o repositório:
   git clone https://github.com/marcella-81/aplicacao-devops.git
cd aplicacao-devops
Instale as dependências com npm install
Rode a aplicação: node src/index.js
Acesse: http://localhost:3000

## Endpoints

Rota:
GET -- Retorna a página de boas-vindas 
GET /integrantes -- Retorna JSON com os integrantes
